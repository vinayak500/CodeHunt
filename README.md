# CodeHunt

A web application where users can sign in and solve algorithmic questions , supporting only the C++ language.
(Kinda like leetcode) but supports only c++ language.

## Features:

- Users can sign up and log in using JSON Web Tokens (JWT) for secure authentication.
- Users can view a list of algorithmic problems.
- C++ code submitted by users can be compiled and executed either locally or using an AWS EC2 instance in a Docker container.
- Users will receive status updates indicating whether their code is correct (Accepted - AC) or incorrect (Wrong Answer - WA).
- User will get notified if there is any compilation or execution errors.
